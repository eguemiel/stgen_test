# stgen_test - Good Hamburger 

Um projeto teste para apresentar para a empresa STgen

# Requirements
* Flutter 3.29.0
* Dart 3.7.0

## Instalation

Clonar o projeto

```bash
  git clone https://github.com/eguemiel/stgen_test.git
```

Instalar dependencias

```bash
flutter pub get
```

Rodar no Windows
```bash
flutter run -d windows
```

Rodar no Linux
```bash
flutter run -d linux
```

Testes
```bash
flutter test
```
